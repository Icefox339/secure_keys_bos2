# Secure Keys BOS2

Android-приложение для Android Studio, написанное на Kotlin и C++ через NDK/JNI. Оно демонстрирует хранение пользовательских файлов в зашифрованном виде и выполнение операций чтения, записи, шифрования и расшифрования в отдельной нативной библиотеке `securefiles`.

## Возможности

- выбор файла через системный Android file picker;
- шифрование и расшифрование выбранного файла;
- парольная аутентификация, смена пароля и несколько пользователей;
- биометрическая аутентификация через `BiometricPrompt` при наличии отпечатка/лица на устройстве;
- хранение файловых ключей: случайный 256-битный ключ пользователя оборачивается AES-GCM мастер-ключом из Android Keystore;
- нативная C++ библиотека выполняет файловый ввод/вывод и криптографическое преобразование через JNI.

## Запуск в Android Studio

1. Откройте папку проекта в Android Studio.
2. Дождитесь синхронизации Gradle и установки Android SDK/NDK/CMake, если IDE предложит это сделать.
3. Запустите конфигурацию `app` на эмуляторе или физическом устройстве с Android 8.0+.
4. При первом запуске задайте пароль пользователя `student`, затем выберите файл для шифрования или расшифрования.

Зашифрованные и расшифрованные результаты сохраняются в private external files директорию приложения и отображаются в статусе на главном экране.
